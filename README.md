This repository contains batch and bash scripts for use with [yt-dlp](https://github.com/yt-dlp/yt-dlp) (which is a fork of youtube-dl) and [ffmpeg](https://github.com/FFmpeg/FFmpeg). I am not affiliated with yt-dlp. 

For some of the files you will see bin\yt-dlp instead of yt-dlp. I think it is better to keep the binary yt-dlp file in a bin folder, separate from the batch or bash scripts in order to avoid accidentally deleting the binary file when selecting a bunch of downloaded files
